## 1. 리액트 사용 이유
### Web-app
Web-app : 앱과 사용성이 비슷한 웹
- 모바일 앱처럼 스무스하게 동작하는 웹들
- 새로고침 없이 웹 탐색이 가능
- React, Angular, Vue를 사용하여 앱 생성
- 모바일 앱으로 발행이 쉬움(PWA, React native)
- 앱처럼 뛰어난 UX
- 비즈니스적 강점
