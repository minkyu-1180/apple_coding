## 1.
### 동적인 UI 만들기
동적인 UI : 유저가 조작 시 형태가 바뀌는 UI
- 모달창, 탭, 서브메뉴, 툴팁, 경고문 등

#### 동적인 UI 만들기 step
1. html, css로 미리 UI 디자인 완성
2. UI의 현재 상태를 state로 저장
3. state에 따라 UI가 어떻게 보일 지 작성
- if-else문은 JSX에서 사용 불가
- 대신 삼항연산자를 사용하기
    - 조건식 ? 조건식true일때실행코드 : 조건식false일때실행코드
